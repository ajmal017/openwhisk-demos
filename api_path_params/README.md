# Shows how to create an API to handle path parameters using OpenWhisk

## Install dependencies
```
npm install
```

## TLDR;
```
npm start
```

## Build
```
npm run build
```

## Deploy
```
npm run deploy
```

## Build and Deploy
```
npm run all
```

## Call API
```
curl https://service.us.apiconnect.ibmcloud.com/gws/apigateway/api/<tenantid>/api/v1/contacts/123
```
